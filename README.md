# 🍽️ Odin Recipes

A simple multi-page recipe website built as part of **The Odin Project – Foundations Course**.

---

## 📖 Overview

Odin Recipes is a basic HTML project that demonstrates how to structure a multi-page website using semantic HTML and proper file organization. The site contains a homepage that links to individual recipe pages, each displaying details about a specific dish.

---

## 🎯 Project Goals

This project was built to practice:

- Writing clean and structured HTML
- Using headings, paragraphs, images, lists, and links
- Creating multi-page websites
- Organizing files and folders effectively
- Working with Git and GitHub for version control

---

## 🧱 Built With

- HTML5
- Git & GitHub
